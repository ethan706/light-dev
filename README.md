> *This repository is under development.*

### Building

1. Get the source code:
```bash
git clone --recursive https://github.com/ethan706/light-dev.git
cd light-dev
```

2. Build with [CMake](https://cmake.org/):
```bash
mkdir build
cd build
cmake ..
cmake --build .
```

> CMake 3.10 & C++17 are required.
